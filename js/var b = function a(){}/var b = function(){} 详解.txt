题目二
（函数声明提前、函数表达式相当于变量赋值所以不会提前、函数声明会覆盖变量声明，但不会覆盖变量赋值）
var a = 1,
  b = function a(x) {
    x &amp;&amp; a(--x);
  };
alert(a);
这个题目的答案是1。
事实上上述代码的相当于下而的代码
var a = 1,
  b = function(x) {
    x &amp;&amp; b(--x);
  };
alert(a);
原题目第二行代码中，b和a同时指向一个地方也就是函数的入口，
但是a和b唯一不同的地方在于函数定义结束也就是};后，a就引用不到了，也就是说a的作用域只在函数体内，而b的作用域却在整个全局范围内。看图说话
