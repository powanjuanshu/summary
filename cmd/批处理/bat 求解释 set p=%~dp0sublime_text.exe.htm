批处理的两句代码不懂，求解释 set p=%~dp0sublime_text.exe set paths=%p:\=\\%

2013-06-11 06:55 liuzenghui2007 | 分类：JAVA相关 
@echo off
echo Windows Registry Editor Version 5.00 > sublime.reg
echo [HKEY_CLASSES_ROOT\*\shell\SublimeText2] >> sublime.reg
echo @="SublimeText2(&W)" >>sublime.reg
echo [HKEY_CLASSES_ROOT\*\shell\SublimeText2\command] >> sublime.reg
set p=%~dp0sublime_text.exe
set paths=%p:\=\\%
echo @="%paths% %%1" >>sublime.reg
sublime.reg&&del sublime.reg
分享到：
2013-06-11 07:08 提问者采纳 
set p=%~dp0sublime_text.exe
设置变量p的值为：批出理文件所在的路径+sublime_text.exe,例如你这个批处理文件放在d:\123\
则变量p的值为:d:\123\sublime_text.exe
set paths=%p:\=\\%
把变量p中的\符号替换成\\,接上面例子，则变量paths的值为:d:\\123\\sublime_text.exe
