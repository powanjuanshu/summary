linux问题！！！请问“~/”是什么意思？比如“打开~/.bash_profile文件”，那么这个文件在哪里？

2013-01-03 11:36 turtlegood | 分类：Linux
分享到：
2013-01-03 11:51
提问者采纳 
~/是当前用户的家目录，在linux下 “~”是/home/yourid/目录的简称，.bash_profile是一个隐藏文件（点开头的都是隐藏文件），这个文件里定义的是该用户的自定义的环境变量。
